### Javascript

Script toevoegen mag, body/html code zelf ni aanpassen.

- Kerstboom (labo 8)
- Formulier uitlezen, berekeningen en feedback geven. DOM manipuleren.

*Er zal altijd een callback aanwezig zijn in de oefening.*
